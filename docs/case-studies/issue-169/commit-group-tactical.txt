    Add group tactical AI system with squad coordination
    
    Implement group behavior for enemies within communication range (1/4 viewport
    diagonal = 360 pixels) based on military tactical principles.
    
    New features:
    - Squad detection and membership tracking within communication range
    - Role assignment system (Leader, Suppressor, Flanker, Assault, Rear Guard)
    - Leader election based on distance to player and health
    - Coordinated suppression fire while squadmates flank
    - Squad communication system with message broadcasting
    - 4 new GOAP actions for squad tactics:
      - ProvideSuppressionAction: Pin player while allies flank
      - CoordinatedFlankAction: Flank under cover of suppression
      - CoordinatedAssaultAction: Synchronized attack when flanker ready
      - CrossCoverAction: Bounding overwatch movement
    
    Technical implementation:
    - Added SquadRole enum with 6 roles
    - Added squad coordination constants and instance variables
    - Integrated squad state into GOAP world state
    - Modified combat, flanking, and assault states for squad behavior
    - Added debug label display for squad role information
    
    Documentation:
    - Added case study documentation in docs/case-studies/issue-99/
    - Includes technical design, timeline reconstruction, and root cause analysis
